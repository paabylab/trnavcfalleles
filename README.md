# About this trnavcfalleles codebase
Scripts and workflows for generating and analyzing information about allelic variation in tRNAs in three Caenorhabditis nematode species. Underlying wild strain sequencing and variant data from [CaeNDR](https://caendr.org/).

For details on the scripts, see the PDF file included here: worm_trna_vcf_allele_code_info.pdf
